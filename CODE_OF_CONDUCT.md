# Code of Conduct

Kocteau should be a calm place to discuss music, taste, product direction, and code.

## Expected Behavior

- Be direct without being cruel.
- Assume good intent, but accept corrections quickly.
- Keep feedback specific and actionable.
- Respect different levels of experience.
- Stay focused on the work, not the person.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks
- Doxxing or sharing private information
- Sexualized, hateful, or discriminatory language
- Repeated off-topic disruption
- Bad-faith arguments that make collaboration harder

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or PRs; block users; or report abuse to GitHub when needed.

If you see a problem, contact the maintainer privately when possible. For security issues, use GitHub private vulnerability reporting.
