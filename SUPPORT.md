# Support

Kocteau is an early public project. Use the right channel so issues stay actionable.

## Bugs

Use the bug report template and include reproduction steps.

## Feature Ideas

Use the feature request template. Keep proposals small and tied to music reviewing, taste discovery, or the For You loop.

## Security

Do not open a public issue. Use GitHub private vulnerability reporting.

## Contribution Help

Start with [CONTRIBUTING.md](./CONTRIBUTING.md). Good first contributions are usually docs, copy, accessibility, loading states, empty states, or isolated web UI polish.
