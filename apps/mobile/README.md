# Kocteau Mobile

This Expo app exists as a placeholder for future mobile work.

Current product focus is the web app in:

```text
apps/web
```

Do not treat `apps/mobile` as production-ready yet. Mobile parity is intentionally deferred until the web core is stronger.

## Future Direction

When mobile work resumes, the likely first pass should reuse the same product assumptions as web:

- OTP-first Supabase Auth
- profile and taste onboarding
- For You as the primary home surface
- review creation/search
- saved reviews and notifications

Until then, web remains the source of truth for product behavior.
